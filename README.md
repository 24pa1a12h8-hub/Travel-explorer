# Travel Explorer

## 401 Course Project

Travel Explorer is a travel experience sharing platform that allows users to explore travel destinations, view traveler reviews, and share their own travel experiences.

## Project Description

Travel Explorer helps travelers discover destinations easily by providing information such as:

- Destination images
- Famous tourist spots nearby
- Top restaurant suggestions
- Traveler reviews and ratings

Users can also submit their own travel experiences and travel tips for other travelers.

---

## Features

- Search for travel destinations
- View destination images and descriptions
- Discover famous places nearby
- Restaurant recommendations
- Traveler reviews section
- Add your own review
- User login and profile
